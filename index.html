Welcome To AlgoMotion

This program is designed to help you visualize the operations of an algorithm taught in a programming data structure. You can insert, update, and delete elements from the algorithms, and see the changes reflected in a graphical representation.

To get started, simply enter a value and an optional index in the input fields below, and click the "Insert" button to add an element to the algorithm. You can also update or delete elements by entering the appropriate index and value, and clicking the corresponding buttons.

As you perform operations on the algorithm, a visual representation will be displayed on the canvas below, showing the current state of the algorithm. Each element will be represented as a box with its value and index displayed.

Feel free to experiment with different operations and see how the algorithm changes. This program is a great way to learn about algorithms and how they work in programming!

Enjoy exploring the world of algorithm & data structures!
