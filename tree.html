Binary Search Tree

TIP Deskripsi Teknis: Binary Search Tree (BST) adalah struktur hierarkis di mana setiap node maksimal memiliki dua anak. Node kiri selalu memiliki nilai lebih kecil dari parent, dan node kanan selalu lebih besar, mendukung pencarian berkecepatan O(log N).
